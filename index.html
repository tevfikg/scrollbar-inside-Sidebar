SCROLLBAR INSIDE SIDEBAR

How to Add a Scrollbar Inside a Sidebar Using HTML and CSS

A sidebar is a common UI element used for navigation, filters, or additional content. When dealing with long content, adding a scrollbar inside the sidebar ensures a better user experience.

Sidebars help organize navigation elements and secondary content, keeping the main content area clean and focused. With responsive design, sidebars can adapt to different screen sizes for optimal viewing on all devices.

Using CSS properties like 'overflow-y: auto' and a fixed height, we can create a scrollable sidebar that contains all navigation options without taking up too much vertical space on the page.

CSS Code for Custom Scrollbar

/* Sidebar Styles */
.sidebar {
    position: fixed;
    right: 20px;
    top: 20px;
    width: 250px;
    height: calc(100vh - 40px);
    background: #e87512;
    color: #eceeee;
    padding: 15px 15px 25px 15px;
    overflow-y: auto;
    scrollbar-width: thin;
    scrollbar-color: #020c56 orangered;
    border-radius: 8px;
    box-shadow: 0 4px 8px rgba(0,0,0,0.2);
    z-index: 100;
}

.sidebar::-webkit-scrollbar {
    width: 8px;
}

.sidebar::-webkit-scrollbar-track {
    background: orangered;
    border-radius: 4px;
}

.sidebar::-webkit-scrollbar-thumb {
    background: #020c56;
    border-radius: 4px;
}

.sidebar::-webkit-scrollbar-thumb:hover {
    background: #010840;
}

.sidebar h2 {
    text-align: center;
    margin-bottom: 20px;
    padding-bottom: 10px;
    border-bottom: 2px solid #eceeee;
}

.sidebar ul {
    list-style-type: none;
    position: relative;
}

.sidebar ul li {
    padding: 12px 0;
    border-bottom: 1px solid rgba(74, 7, 132, 0.3);
}

.sidebar ul li:last-child {
    border-bottom: none;
}

.sidebar ul li a {
    color: #eceeee;
    text-decoration: none;
    display: block;
    transition: padding 0.3s ease;
}

.sidebar ul li a:hover {
    padding-left: 10px;
    font-weight: bold;
}

/* Scroll indicator */
.sidebar::after {
    content: "↓ Scroll for more ↓";
    position: sticky;
    bottom: 0;
    left: 0;
    width: 100%;
    background: linear-gradient(transparent, #e87512 40%);
    text-align: center;
    padding: 15px 0 5px 0;
    font-weight: bold;
    display: block;
    margin-top: 10px;
    animation: pulse 2s infinite;
}

@keyframes pulse {
    0% { opacity: 0.7; }
    50% { opacity: 1; }
    100% { opacity: 0.7; }
}

/* Partially visible items at the bottom */
.sidebar ul::after {
    content: "";
    position: sticky;
    bottom: -15px;
    left: 0;
    height: 20px;
    width: 100%;
    background: linear-gradient(transparent, #e87512);
    pointer-events: none;
}
